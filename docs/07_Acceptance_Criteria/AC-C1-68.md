# Acceptance Criteria: C1-68

**Source:** C1-68 - [STORY-3.2.1] As an Admin, I want to filter and view leads/dashboards by Business Category and Sub-Category so that I can identify high-value segments to target....
**Issue Type:** Story

## Criteria

Given an Admin selects a Category filter on the Dashboard, then all dashboard cards and charts recalculate to reflect only leads in that Category (and its sub-categories, if also filtered).

Given the Won-rate-by-Category widget, then it displays (Won leads / Total closed leads) per category as a percentage, updated in real time as leads change stage.

Given a CSV/Excel export is run, then Category and Sub-Category are included as columns in the output file.
