# Acceptance Criteria: C1-122

**Source:** C1-122 - [STORY-6.3.1] As an Admin, I want to export lead and follow-up data so that I can run analysis outside the CRM....
**Issue Type:** Story

## Criteria

Given an Admin applies filters on Lead List and clicks Export, then the exported file contains exactly the filtered result set, no more and no fewer rows.

Given an export completes, then an Audit Log entry is created recording the actor, record count, filter criteria, and timestamp.

Marketing Executives do not see the Export action anywhere in the UI.
