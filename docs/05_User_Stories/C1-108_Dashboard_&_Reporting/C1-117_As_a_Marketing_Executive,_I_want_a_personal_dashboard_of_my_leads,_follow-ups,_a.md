## Description

User Story:

As a Marketing Executive, I want a personal dashboard of my leads, follow-ups, and results so that I can manage my own pipeline.

Acceptance Criteria:

Given a Marketing Executive opens their dashboard, then every figure shown reflects only leads where Assigned To = that user — verified at the API/query layer, not merely filtered in the browser.

Given the 'My Follow-ups Today' card, then clicking it navigates directly to a pre-filtered Lead List showing exactly today's due follow-ups for that user.
